# ChatAI – MERN Stack Gemini AI Chat Application

ChatAI is a modern AI-powered chat application built with the MERN Stack and integrated with the Google Gemini API.  
Users can interact with an AI assistant in real time, ask questions, and receive intelligent responses.

## 🌐 Live Preview
https://chat-ai-pearl-kappa.vercel.app/

## Features

- AI Chat powered by Google Gemini API
- Real-time conversation interface
- Modern and responsive dark & light mode UI
- Full-stack MERN architecture
- Fast API responses
- Clean and simple chat experience
- Secure backend handling

## Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Axios
- lucide-react icons

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB

**AI Integration**
- Google Gemini API

**Deployment**
- Vercel
